- Le client demande "Paris" → il récupère {"type":"loading"} immédiatement
- En parallèle, MeteoService.fetch() tourne en tâche de fond
- Quand le fetch aboutit, callbacks.update([...]) est appelé
- Stocke la valeur dans storedResults puis appelle invalidateLazyKey
- Skip propage automatiquement : TLazyDir sait que CityMapper a lu cette clé, donc il re-déclenche mapEntry pour Paris
- mapEntry refait meteo.getArray("Paris") → computeFor → getStoredResult → trouve la valeur → renvoie {"type":"value",...}
- La EagerCollection cities est mise à jour avec la nouvelle valeur pour Paris
- Le prochain snapshot du client récupère la value sans aucun re-fetch côté API

WORKFLOW EXAMPLE AVEC UNE API METEO

Quelqu'un demande Paris - sens ascendant



Utilisateur demande "Paris"
      ↓
LazyCollection.getArray("Paris")          ← index.ts (LazyCollectionImpl)
      ↓ cache vide
computeFor("Paris") déclenché             ← Runtime.sk (ExternalServiceLazyCompute)
      ↓
service.fetch("mock", "Paris") appelé     ← Runtime.sk → FFI.sk → index.ts (ServiceDefinition)
      ↓
addSubscription(context, waitable)        ← Runtime.sk
      ↓
retourne { type: "loading" }              ← Runtime.sk
      ↓
Utilisateur reçoit { type: "loading" }    ← index.ts (LazyCollectionImpl)

-------------------------------------------------------------------------------------------

Fetch renvoie 18°C



LazyExternalService.fetch complète        ← adaptateur (MockLazyService)
      ↓
callbacks.update(["18°C"]) appelé         ← index.ts (ServiceDefinition)
      ↓
ServiceDefinition stocke "Paris" → "18°C" ← index.ts (Map interne)
      ↓
invalidateLazyKey("Paris") appelé         ← index.ts → binding.ts → FFI.sk → Runtime.sk
      ↓
Cache de TLazyDir vidé pour "Paris"       ← Runtime.sk (invalidateLazyKey)

-------------------------------------------------------------------------------------------

Quelqu'un re-demande Paris



Utilisateur redemande "Paris"
      ↓
LazyCollection.getArray("Paris")          ← index.ts (LazyCollectionImpl)
      ↓ cache vide (invalidé)
computeFor("Paris") déclenché             ← Runtime.sk (ExternalServiceLazyCompute)
      ↓
service.getStoredResult("mock", "Paris")  ← Runtime.sk → FFI.sk → index.ts (ServiceDefinition)
      ↓ valeur trouvée dans le Map
retourne { type: "value", payload: "18°C" } ← Runtime.sk
      ↓
Utilisateur reçoit { type: "value", payload: "18°C" } ← index.ts
